# Crete a Project

This example will create a new cluster GKE. This is the minimal configuration 
for having a working Kubernetes cluster.

## Expected Variables

- `name`
- `project_id`
- `zone`
- `network`
- `subnetwork`
- `pods_ip_range`
- `services_ip_range`
- `master_ip_range`
- `master_authorized_networks_config`

- `node_pool_name`
- `node_pool_node_count`
- `node_pool_node_type`
- `service_account_email`
